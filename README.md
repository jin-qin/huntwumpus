# Hunt the Wumpus

## Prerequisites
1. CMake latest release
2. make tool (for Linux and MacOS) or visual studio(for Windows)

## Build
1. Linux & MacOS
    ```sh
    mkdir build
    cd build && cmake ..
    make
    ```
2. Windows
   ```sh
   mkdir build
   cd build && cmake ..
   ```
   `open solution with your visual studio`
